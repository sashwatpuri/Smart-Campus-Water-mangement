# Water Quality Prediction Model

This is the water quality prediction model which will be used in the water cooler that our university has to monitor the parameters for healthy drinking water.

## How to Run the Model

To run the application, execute the following command:

```bash
python app.py
```

## How to Open the Dashboard

1. Download the `dashboard.html` file from the project
2. Double-click the `dashboard.html` file to open it in your web browser
3. The dashboard will display the water quality predictions and parameters
